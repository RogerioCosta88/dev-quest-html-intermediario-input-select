Select

A Tag select é usada para que o usuário escolha um ou mais opções pré definidas em uma lista.

O select possui abertura e fechamento.

Option

É inserida dentro da Tag select e é onde inserimos as opções que seram apresentadas para o usuário.

Atributos

Value

É definida no HTML e é o que será enfiado para o backend e/ou frontend quando o usuário clicar em submit

Multiple

Permite que o usuário escolha mais de uma opção. Para selecionar mais de uma opção o usuário deve segurar a tecla Ctrl e clicar nas outras opções desejadas. Esse atributo é colocado dentro da Tag Select.
